# Introduktion till Kubernetes

En praktisk introduktionskurs i Kubernetes med Rancher Kubernetes Enginge (RKE2).

---

## Översikt

Kursen täcker Kubernetes grundläggande koncept och ger deltagarna praktisk erfarenhet av att driftsätta och hantera workloads i ett kluster.

**Längd:** ~2 timmar
**Förkunskaper:** Grundläggande kännedom om Docker och containers

---
